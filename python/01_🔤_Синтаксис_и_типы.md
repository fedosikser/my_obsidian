## Ключевое отличие от C++
Если ты пришел из C++, забудь на время про фигурные скобки `{}`, точки с запятой `;` и строгие типы.
*   **Отступы решают всё:** Блоки кода определяются не `{}`, а **уровнем отступа** (обычно 4 пробела). Одинаковые отступы — один блок. Это не просто стиль, это закон.
*   **Динамическая типизация:** Переменной не назначают тип. Тип определяется автоматически тем значением, которое в неё положили.
*   **Всё — объект:** Да, абсолютно всё: числа, строки, функции и даже классы.

## Переменные
Создание переменной — это просто присваивание значения. Никаких `int`, `string` и т.д.

```python
# Имена переменных создаются в момент первого присваивания
name = "Алиса"          # Текст (str)
age = 30                # Целое число (int)
height = 1.75           # Число с плавающей точкой (float)
is_student = True       # Логическое значение (bool) - обязательно с большой буквы!
nothing = None          # Отсутствие значения (аналог NULL)

# Множественное присваивание
x, y, z = 1, 2, 3       # x=1, y=2, z=3
a = b = c = 0           # a=0, b=0, c=0

# В Python нет констант в классическом понимании.
# По договоренности, имена "констант" пишут ЗАГЛАВНЫМИ_БУКВАМИ.
PI = 3.14159
SPEED_OF_LIGHT = 299792458
```

## Базовые типы и операции

### Числа
Python поддерживает целые (`int`), вещественные (`float`) и комплексные (`complex`) числа.

```python
# int — целые числа неограниченной длины (хоть 100 цифр!)
x = 42
big_number = 10**100  # 1 с 100 нулями

# float — числа с плавающей точкой
y = 3.14159
sci = 1.5e-10          # Научная нотация: 0.00000000015

# Операции
sum = 10 + 5           # 15
diff = 10 - 5          # 5
product = 10 * 5       # 50

# ВНИМАНИЕ: Деление!
quotient = 10 / 3      # 3.3333333333333335 (обычное деление -> float)
int_quotient = 10 // 3 # 3 (целочисленное деление)
remainder = 10 % 3     # 1 (остаток от деления)

power = 2 ** 3         # 8 (возведение в степень)

# Встроенные функции
abs(-5)                # 5 (модуль числа)
round(3.14159, 2)      # 3.14 (округление до 2 знаков)
pow(2, 3)              # 8 (аналог 2**3)
```

### Строки (str)
Строки можно создавать с разными кавычками. Главная фишка Python — удобные **срезы** и **f-строки**.

```python
# Три способа создать строку
s1 = 'Одинарные кавычки'
s2 = "Двойные кавычки"
s3 = """Многострочный
текст, который может
занимать несколько строк"""

# Конкатенация и повторение
hello = "Привет" + " " + "мир"   # "Привет мир"
ha = "ха" * 3                    # "хахаха"

# Длина строки
len(hello)                       # 10

# Основные методы (важно!)
text = "  Hello, World!  "
text.upper()                     # "  HELLO, WORLD!  "
text.lower()                     # "  hello, world!  "
text.strip()                      # "Hello, World!" (удаляет пробелы по краям)
text.replace("World", "Python")   # "  Hello, Python!  "

# Разделение и объединение
"яблоко,банан,вишня".split(",")   # ["яблоко", "банан", "вишня"]
", ".join(["a", "b", "c"])        # "a, b, c"

# Проверки
"123".isdigit()                    # True (только цифры)
"abc".isalpha()                    # True (только буквы)
"abc123".isalnum()                 # True (буквы и цифры)
"Hello".startswith("He")           # True
"Hello".endswith("lo")             # True

# Поиск
text = "Hello, World!"
"World" in text                    # True (проверка вхождения)
text.find("World")                  # 7 (индекс начала подстроки)
text.count("o")                     # 2 (количество вхождений)

# Индексы и срезы (гениальная вещь!)
s = "Python"
s[0]      # 'P'  (первый символ)
s[1]      # 'y'  (второй)
s[-1]     # 'n'  (ПОСЛЕДНИЙ символ! Отрицательные индексы работают!)
s[-2]     # 'o'  (предпоследний)

# Срезы: [start:stop:step]
s[1:4]    # 'yth' (символы с индекса 1 по 3, не включая 4)
s[:3]     # 'Pyt' (от начала до индекса 3)
s[3:]     # 'hon' (от индекса 3 до конца)
s[::2]    # 'Pto' (каждый второй)
s[::-1]   # 'nohtyP' (разворот строки - очень полезно!)
```

### Форматирование строк (f-strings)
Это современный и самый удобный способ вставлять переменные в строки (доступен с Python 3.6+).

```python
name = "Алиса"
age = 30
pi = 3.14159

# Буква f перед строкой ВКЛЮЧАЕТ режим подстановки
message = f"Меня зовут {name}, мне {age} лет."
print(message)  # Меня зовут Алиса, мне 30 лет.

# Внутри {} можно писать выражения
print(f"Через 5 лет мне будет {age + 5}.")
print(f"2 в 10 степени = {2**10}.")

# Форматирование чисел
print(f"Число пи с двумя знаками: {pi:.2f}")   # 3.14
print(f"Большое число: {1000000:,}")           # 1,000,000 (разделители тысяч)
```

### Булевы значения (bool)
Всего два значения: `True` и `False`. Обрати внимание на заглавные буквы.

```python
is_ready = True
is_finished = False

# Логические операторы - пишутся СЛОВАМИ!
result = True and False    # False (аналог &&)
result = True or False     # True  (аналог ||)
result = not True          # False (аналог !)

# Сравнения
x = 5
print(x > 3)      # True
print(x == 5)     # True (Обрати внимание: ==, а не =)
print(x != 10)    # True

# Классная фишка Python: цепочки сравнений
print(1 < x < 10)  # True (в C++ такое не скомпилируется!)
```

### Приведение типов
Явное преобразование из одного типа в другой.

```python
# В строку
str(123)          # "123"
str(3.14)         # "3.14"
str(True)         # "True"

# В число
int("123")        # 123
int(3.9)          # 3 (отбрасывает дробную часть, не округляет!)
float("3.14")     # 3.14
float("1e-3")     # 0.001

# В булево значение
bool(1)           # True
bool(0)           # False
bool("")          # False (пустая строка)
bool("текст")     # True (непустая строка)
bool([])          # False (пустой список)
bool([1, 2])      # True (непустой список)

# Правило: в False превращаются:
# - 0, 0.0, 0j
# - Пустые коллекции: "", [], (), {}, set()
# - None
# Всё остальное — True.
```

## Полезные мелочи

### Множественное сравнение (очень Pythonic!)
```python
age = 25
if 18 <= age <= 65:
    print("Трудоспособный возраст")
```

### Обмен значений (без временной переменной!)
```python
a, b = 10, 20
a, b = b, a      # Теперь a=20, b=10
```

### Проверка на None
```python
value = None
if value is None:      # Используй 'is', а не '=='
    print("Нет значения")
if value is not None:
    print("Что-то есть")
```
