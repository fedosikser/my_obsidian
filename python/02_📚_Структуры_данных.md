# 02_📚_Структуры_данных

## Общая концепция
В Python все структуры данных можно разделить на:
- **Изменяемые (mutable)** — можно менять после создания
- **Неизменяемые (immutable)** — нельзя изменить после создания

| Тип | Изменяемый | Упорядоченный | Индексация | Дубликаты |
|-----|------------|---------------|------------|-----------|
| list | ✅ | ✅ | ✅ | ✅ |
| tuple | ❌ | ✅ | ✅ | ✅ |
| dict | ✅ | ✅ (3.7+) | по ключу | ключи уникальны |
| set | ✅ | ❌ | ❌ | ❌ |
| str | ❌ | ✅ | ✅ | ✅ |

## Список (list) — аналог std::vector

### Создание
```python
# Разные способы создания
empty = []                          # Пустой список
numbers = [1, 2, 3, 4, 5]          # С числами
mixed = [1, "два", 3.0, [4, 5]]    # Можно смешивать типы!
from_range = list(range(5))         # [0, 1, 2, 3, 4]
from_string = list("hello")         # ['h', 'e', 'l', 'l', 'o']
repeated = [0] * 5                  # [0, 0, 0, 0, 0]
```

### Индексация и срезы (как у строк)
```python
lst = [10, 20, 30, 40, 50]

# Доступ по индексу
lst[0]        # 10 (первый)
lst[2]        # 30
lst[-1]       # 50 (последний!)
lst[-2]       # 40 (предпоследний)

# Срезы [start:stop:step]
lst[1:4]      # [20, 30, 40] (с 1 по 3)
lst[:3]       # [10, 20, 30] (с начала до индекса 3)
lst[3:]       # [40, 50] (с индекса 3 до конца)
lst[::2]      # [10, 30, 50] (каждый второй)
lst[::-1]     # [50, 40, 30, 20, 10] (разворот)
```

### Основные методы
```python
lst = [1, 2, 3]

# Добавление
lst.append(4)              # [1, 2, 3, 4] (в конец)
lst.insert(1, 1.5)         # [1, 1.5, 2, 3, 4] (по индексу)
lst.extend([5, 6])         # [1, 1.5, 2, 3, 4, 5, 6] (объединить)

# Удаление
lst.remove(1.5)            # удалить ПЕРВОЕ вхождение 1.5
last = lst.pop()           # удалить и вернуть последний (6)
second = lst.pop(1)        # удалить и вернуть элемент с индексом 1
del lst[0]                 # удалить по индексу (без возврата)
lst.clear()                # очистить весь список []

# Поиск и информация
lst = [10, 20, 30, 20, 40]
lst.index(20)              # 1 (индекс ПЕРВОГО вхождения)
lst.index(20, 2)           # 3 (искать с индекса 2)
lst.count(20)              # 2 (количество вхождений)
len(lst)                   # 5 (длина списка)
20 in lst                  # True (проверка вхождения)

# Сортировка и разворот
lst.sort()                 # сортировка (изменяет список)
lst.sort(reverse=True)     # по убыванию
lst.reverse()              # разворот (изменяет список)

# Не изменяют оригинал
sorted(lst)                # вернуть отсортированную копию
list(reversed(lst))        # вернуть развернутую копию
```

### Перебор списка
```python
fruits = ["яблоко", "банан", "вишня"]

# По элементам
for fruit in fruits:
    print(fruit)

# По индексам
for i in range(len(fruits)):
    print(f"{i}: {fruits[i]}")

# С enumerate (самый питоничный способ!)
for i, fruit in enumerate(fruits):
    print(f"{i}: {fruit}")

# С enumerate отличным от 0
for i, fruit in enumerate(fruits, start=1):
    print(f"{i}. {fruit}")
```

### List Comprehension (генератор списков) — ОЧЕНЬ ВАЖНО!
Это одна из самых красивых фишек Python. Заменяет циклы с условиями в одну строку.

```python
# Вместо:
squares = []
for x in range(10):
    squares.append(x ** 2)
# [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

# Пишем:
squares = [x ** 2 for x in range(10)]

# С условием (только четные)
evens = [x for x in range(10) if x % 2 == 0]  # [0, 2, 4, 6, 8]

# С тернарным оператором
["чет" if x % 2 == 0 else "нечет" for x in range(5)]  # ['чет', 'нечет', 'чет', 'нечет', 'чет']

# Вложенные циклы
pairs = [(x, y) for x in [1, 2] for y in [3, 4]]
# [(1, 3), (1, 4), (2, 3), (2, 4)]

# Сложные преобразования
words = ["hello", "world", "python"]
uppercase = [word.upper() for word in words]  # ['HELLO', 'WORLD', 'PYTHON']
lengths = [len(word) for word in words]       # [5, 5, 6]
```

### Копирование списков (важно!)
```python
a = [1, 2, 3]

# Неправильно (это ссылка, не копия!)
b = a
b.append(4)
print(a)  # [1, 2, 3, 4] — исходный тоже изменился!

# Правильные способы копирования
b = a.copy()                # метод copy()
b = list(a)                 # через конструктор
b = a[:]                    # через срез (самый быстрый)
b = [x for x in a]          # через генератор

# Для вложенных списков нужна глубокая копия
import copy
nested = [[1, 2], [3, 4]]
deep_copy = copy.deepcopy(nested)
```

## Кортеж (tuple) — неизменяемый список

### Создание
```python
# Разные способы
empty = ()                         # Пустой кортеж
point = (10, 20)                   # Кортеж из двух элементов
singleton = (42,)                  # ВНИМАНИЕ: обязательная запятая!
without_parentheses = 1, 2, 3      # Тоже кортеж! (упаковка)
from_list = tuple([1, 2, 3])       # Из списка
from_string = tuple("abc")         # ('a', 'b', 'c')
```

### Использование
```python
point = (10, 20)

# Доступ (как к списку)
point[0]          # 10
point[1]          # 20
# point[0] = 5    # ОШИБКА! Нельзя изменять

# Распаковка кортежей (tuple unpacking)
x, y = point                     # x=10, y=20
x, y = y, x                      # Обмен значений (без временной!)
a, b, *rest = (1, 2, 3, 4, 5)    # a=1, b=2, rest=[3, 4, 5]
first, *middle, last = (1, 2, 3, 4)  # first=1, middle=[2,3], last=4

# Кортежи как ключи словарей (можно!)
locations = {
    (40.7128, -74.0060): "Нью-Йорк",
    (51.5074, -0.1278): "Лондон"
}
```

### Зачем нужны кортежи?
1. **Безопасность** — данные нельзя случайно изменить
2. **Быстрее списков** — неизменяемость оптимизирует память
3. **Ключи для словарей** — только неизменяемые типы
4. **Возврат нескольких значений из функции**

```python
def get_stats(numbers):
    """Функция возвращает кортеж"""
    return min(numbers), max(numbers), sum(numbers)/len(numbers)

minimum, maximum, average = get_stats([1, 2, 3, 4, 5])
```

## Словарь (dict) — аналог unordered_map/std::map

### Создание
```python
# Разные способы
empty = {}                         # Пустой словарь
user = {
    "name": "Алиса",
    "age": 30,
    "city": "Москва"
}

from_pairs = dict([("name", "Боб"), ("age", 25)])
with_keys = dict(name="Чарли", age=35)  # Только для строковых ключей!
from_zip = dict(zip(["x", "y"], [10, 20]))  # {'x': 10, 'y': 20}
```

### Доступ и операции
```python
user = {"name": "Алиса", "age": 30}

# Чтение
name = user["name"]          # "Алиса"
# phone = user["phone"]      # ОШИБКА! ключа нет

# Безопасное чтение (рекомендуется!)
phone = user.get("phone")                # None (не ошибка!)
phone = user.get("phone", "не указан")   # "не указан" (значение по умолчанию)

# Добавление/изменение
user["job"] = "Инженер"      # Добавить новую пару
user["age"] = 31             # Изменить существующую

# Проверка наличия
if "name" in user:
    print("Имя есть!")

# Удаление
del user["city"]              # Удалить ключ (если есть)
age = user.pop("age")         # Удалить и вернуть значение
last = user.popitem()         # Удалить и вернуть последнюю добавленную пару

# Длина
len(user)                     # Количество пар
```

### Перебор словаря
```python
user = {"name": "Алиса", "age": 30, "city": "Москва"}

# По ключам (2 способа)
for key in user:
    print(key)

for key in user.keys():
    print(key)

# По значениям
for value in user.values():
    print(value)

# По парам (САМЫЙ ПОПУЛЯРНЫЙ!)
for key, value in user.items():
    print(f"{key}: {value}")
```

### Словарные генераторы (dict comprehension)
```python
# Квадраты чисел
squares = {x: x**2 for x in range(5)}
# {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

# С условием
even_squares = {x: x**2 for x in range(10) if x % 2 == 0}
# {0: 0, 2: 4, 4: 16, 6: 36, 8: 64}

# Инвертирование словаря
original = {"a": 1, "b": 2, "c": 3}
inverted = {value: key for key, value in original.items()}
# {1: 'a', 2: 'b', 3: 'c'}

# Из двух списков
keys = ["name", "age", "city"]
values = ["Алиса", 30, "Москва"]
user = {keys[i]: values[i] for i in range(len(keys))}
# Или лучше (более питонично):
user = dict(zip(keys, values))
```

### Полезные методы
```python
# Объединение словарей (Python 3.9+)
dict1 = {"a": 1, "b": 2}
dict2 = {"b": 3, "c": 4}
merged = dict1 | dict2           # {'a': 1, 'b': 3, 'c': 4}
dict1 |= dict2                   # обновить dict1

# Старый способ
merged = {**dict1, **dict2}      # распаковка
merged = dict1.copy()
merged.update(dict2)

# Значения по умолчанию
from collections import defaultdict

# defaultdict сам создает отсутствующие ключи
word_count = defaultdict(int)     # int() дает 0
for word in ["a", "b", "a", "c"]:
    word_count[word] += 1         # ключи создаются автоматически!

group_by_length = defaultdict(list)
for word in ["cat", "dog", "elephant"]:
    group_by_length[len(word)].append(word)
# {3: ['cat', 'dog'], 7: ['elephant']}
```

## Множество (set) — уникальные значения

### Создание
```python
# Разные способы
empty = set()                    # Пустое множество (не {}!)
numbers = {1, 2, 3, 3, 3}        # {1, 2, 3} (дубликаты удалены)
from_list = set([1, 2, 2, 3])    # {1, 2, 3}
from_string = set("hello")       # {'h', 'e', 'l', 'o'} (порядок случаен)

# ВНИМАНИЕ! {} — это пустой СЛОВАРЬ, не множество!
wrong = {}                       # dict, не set!
```

### Операции с элементами
```python
s = {1, 2, 3}

# Добавление
s.add(4)                         # {1, 2, 3, 4}
s.add(2)                         # {1, 2, 3, 4} (без изменений)

# Удаление
s.remove(3)                      # {1, 2, 4} (ошибка если нет)
s.discard(10)                    # {1, 2, 4} (ошибки НЕТ!)
x = s.pop()                       # удалить и вернуть случайный элемент
s.clear()                         # очистить множество
```

### Теоретико-множественные операции
```python
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

# Объединение
a | b                            # {1, 2, 3, 4, 5, 6}
a.union(b)                       # то же самое

# Пересечение
a & b                            # {3, 4}
a.intersection(b)                # то же самое

# Разность
a - b                            # {1, 2} (что есть в a, но нет в b)
b - a                            # {5, 6} (что есть в b, но нет в a)
a.difference(b)                  # то же самое

# Симметрическая разность
a ^ b                            # {1, 2, 5, 6} (в одном или другом, но не в обоих)
a.symmetric_difference(b)        # то же самое

# Проверки
a.issubset({1, 2, 3, 4, 5})      # True (a подмножество?)
a.issuperset({1, 2})             # True (a надмножество?)
a.isdisjoint({5, 6})             # True (нет общих элементов?)
```

### Множественные генераторы (set comprehension)
```python
# Квадраты четных чисел
even_squares = {x**2 for x in range(10) if x % 2 == 0}
# {0, 4, 16, 36, 64}

# Уникальные буквы в словах
words = ["hello", "world", "python"]
unique_letters = {letter for word in words for letter in word}
# {'h', 'e', 'l', 'o', 'w', 'r', 'd', 'p', 'y', 't', 'n'}
```

### Замороженное множество (frozenset) — неизменяемый set
```python
# Можно использовать как ключ словаря
fs = frozenset([1, 2, 3])
# fs.add(4)                     # ОШИБКА! нельзя изменить

d = {fs: "это работает"}        # frozenset можно
# d = {{1,2}: "ошибка"}         # set нельзя (неизменяемый нужен)
```

## Сравнение с C++

| C++ | Python | Примечание |
|-----|--------|------------|
| `std::vector` | `list` | Динамический массив |
| `std::array` | `tuple` | Фиксированный размер |
| `std::map`/`unordered_map` | `dict` | Ассоциативный массив |
| `std::set`/`unordered_set` | `set` | Множество |
| `std::pair` | `tuple` из 2 | Пара значений |
| `std::stack`/`queue` | `list` + методы | Или `collections.deque` |
