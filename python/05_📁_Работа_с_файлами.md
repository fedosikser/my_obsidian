# 05_📁_Работа_с_файлами

## Философия работы с файлами в Python
- **Контекстный менеджер `with`** — всегда используй его!
- **Явное лучше неявного** — режимы открытия указываются явно
- **Разные режимы** для текста и бинарных данных
- **Кроссплатформенность** — Python сам обрабатывает различия ОС

---

## Открытие и закрытие файлов

### Правильный способ (с `with`)
```python
# with автоматически закроет файл после выхода из блока
with open("file.txt", "r", encoding="utf-8") as file:
    content = file.read()
    # файл закроется автоматически здесь
```

### Неправильный способ (так не делай!)
```python
# Можно забыть закрыть!
file = open("file.txt", "r")
content = file.read()
file.close()  # если забудешь — утечка памяти
```

## Режимы открытия файлов

| Режим | Описание | Если файла нет | Позиция |
|-------|----------|----------------|---------|
| `"r"` | Чтение (по умолчанию) | Ошибка | Начало |
| `"w"` | Запись (перезаписывает) | Создается | Начало |
| `"a"` | Добавление в конец | Создается | Конец |
| `"x"` | Эксклюзивное создание | Создается | Начало |
| `"r+"` | Чтение и запись | Ошибка | Начало |
| `"w+"` | Чтение и запись (перезапись) | Создается | Начало |
| `"a+"` | Чтение и запись (добавление) | Создается | Конец |

**Важно добавить `"b"` для бинарных файлов:**
- `"rb"` — чтение бинарного файла
- `"wb"` — запись бинарного файла
- `"ab"` — добавление в бинарный файл

### Примеры
```python
# Текстовые режимы
with open("file.txt", "r") as f:      # чтение
with open("file.txt", "w") as f:      # запись
with open("file.txt", "a") as f:      # добавление

# Бинарные режимы
with open("image.jpg", "rb") as f:    # чтение бинарника
with open("image.jpg", "wb") as f:    # запись бинарника

# Чтение и запись
with open("file.txt", "r+") as f:     # чтение и запись
```

## Чтение файлов

### Чтение целиком
```python
# Весь файл в строку
with open("file.txt", "r", encoding="utf-8") as f:
    content = f.read()
    print(content)

# Все строки в список
with open("file.txt", "r", encoding="utf-8") as f:
    lines = f.readlines()
    print(lines)  # ['строка1\n', 'строка2\n', ...]
```

### Построчное чтение (для больших файлов!)
```python
# Самый эффективный способ для больших файлов
with open("large_file.txt", "r", encoding="utf-8") as f:
    for line in f:  # читает по одной строке, не грузит всё в память
        print(line.strip())  # strip() убирает \n

# Чтение по кускам (если нет перевода строк)
with open("large_file.bin", "rb") as f:
    while chunk := f.read(8192):  # читаем по 8KB
        process(chunk)  # обрабатываем кусок
```

### Чтение порциями
```python
# Первые N символов
with open("file.txt", "r") as f:
    header = f.read(100)  # первые 100 символов

# Чтение до определенного символа
with open("file.txt", "r") as f:
    line = f.readline()  # одна строка
    rest = f.read()      # всё остальное
```

### Позиционирование
```python
with open("file.txt", "r") as f:
    print(f.tell())      # 0 (текущая позиция)
    f.read(10)
    print(f.tell())      # 10
    f.seek(0)            # вернуться в начало
    f.seek(10, 1)        # на 10 от текущей
    f.seek(-5, 2)        # на 5 от конца (только для бинарных)
```

## Запись в файлы

### Простая запись
```python
# Запись строки
with open("out.txt", "w", encoding="utf-8") as f:
    f.write("Привет, мир!\n")
    f.write("Вторая строка\n")

# Запись списка строк
lines = ["строка1\n", "строка2\n", "строка3\n"]
with open("out.txt", "w") as f:
    f.writelines(lines)  # записывает все строки
```

### Добавление в конец
```python
with open("log.txt", "a", encoding="utf-8") as f:
    f.write(f"Новая запись: {datetime.now()}\n")
```

### Форматированный вывод
```python
data = [
    {"name": "Анна", "age": 30},
    {"name": "Боб", "age": 25}
]

with open("output.txt", "w") as f:
    for item in data:
        f.write(f"Имя: {item['name']}, Возраст: {item['age']}\n")
```

## Работа с путями (pathlib) — СОВРЕМЕННЫЙ СПОСОБ!

### Path — основной класс
```python
from pathlib import Path

# Создание путей
p = Path("/home/user/docs/file.txt")
p = Path("folder") / "subfolder" / "file.txt"  # через /
p = Path.home() / "documents" / "file.txt"     # домашняя директория
p = Path.cwd() / "data" / "config.json"        # текущая директория

# Свойства пути
p.name          # 'file.txt' (имя файла)
p.stem          # 'file' (имя без расширения)
p.suffix        # '.txt' (расширение)
p.parent        # /home/user/docs (родительская папка)
p.parents       # [Path('/home/user/docs'), Path('/home/user'), Path('/home'), Path('/')]
p.parts         # ('/', 'home', 'user', 'docs', 'file.txt')
p.absolute()    # абсолютный путь
```

### Проверки
```python
p = Path("file.txt")

p.exists()      # существует ли?
p.is_file()     # это файл?
p.is_dir()      # это папка?
p.is_absolute() # это абсолютный путь?

# Статистика
p.stat()        # полная информация
p.stat().st_size  # размер в байтах
p.stat().st_mtime # время изменения
```

### Операции с файлами
```python
p = Path("test.txt")

# Чтение/запись (удобно!)
p.write_text("Hello, World!", encoding="utf-8")
content = p.read_text(encoding="utf-8")

p.write_bytes(b"binary data")
data = p.read_bytes()

# Создание/удаление
p.touch()                       # создать пустой файл
p.rename("new.txt")              # переименовать
p.replace("new.txt")             # заменить/переместить
p.unlink()                       # удалить файл
p.symlink_to("target.txt")       # создать симлинк
```

### Операции с директориями
```python
d = Path("new_folder")

# Создание
d.mkdir()                        # создать папку
d.mkdir(parents=True)             # создать все родительские
d.mkdir(parents=True, exist_ok=True)  # не падать, если есть

# Содержимое
list(d.iterdir())                 # все файлы и папки
list(d.glob("*.txt"))             # все .txt файлы
list(d.rglob("*.py"))             # рекурсивно все .py

# Удаление
d.rmdir()                         # удалить пустую папку

# Рекурсивное удаление (осторожно!)
import shutil
shutil.rmtree(d)                  # удалить папку со всем содержимым
```

## Копирование, перемещение, удаление (shutil)

### Копирование
```python
import shutil

# Копирование файла
shutil.copy("src.txt", "dst.txt")           # копирует содержимое
shutil.copy2("src.txt", "dst.txt")          # + метаданные

# Копирование папки
shutil.copytree("src_dir", "dst_dir")       # рекурсивно
shutil.copytree("src_dir", "dst_dir", dirs_exist_ok=True)  # Python 3.8+
```

### Перемещение/переименование
```python
# Перемещение файла или папки
shutil.move("file.txt", "archive/file.txt")
shutil.move("folder", "backup/folder")

# Переименование (можно и os.rename)
Path("old.txt").rename("new.txt")
```

### Удаление
```python
# Файл
Path("file.txt").unlink()
os.remove("file.txt")  # старый способ

# Пустая папка
Path("empty_dir").rmdir()
os.rmdir("empty_dir")

# Папка с содержимым
shutil.rmtree("folder_with_stuff")
```

## Старый способ (os.path) — для совместимости

### os.path
```python
import os

# Сборка путей
path = os.path.join("folder", "subfolder", "file.txt")

# Разбор
os.path.basename(path)      # 'file.txt'
os.path.dirname(path)       # 'folder/subfolder'
os.path.split(path)         # ('folder/subfolder', 'file.txt')
os.path.splitext(path)      # ('folder/subfolder/file', '.txt')

# Проверки
os.path.exists(path)
os.path.isfile(path)
os.path.isdir(path)
os.path.getsize(path)
os.path.getmtime(path)

# Абсолютный путь
os.path.abspath("file.txt")
```

### os — операции с файловой системой
```python
import os

# Текущая директория
os.getcwd()
os.chdir("/path/to/dir")

# Список файлов
os.listdir(".")
os.listdir("/path")

os.mkdir("new_dir")
os.makedirs("a/b/c", exist_ok=True)

# Удаление
os.remove("file.txt")
os.rmdir("empty_dir")        # только пустые
```

## Временные файлы и папки
```python
import tempfile

# Временный файл (автоматически удалится)
with tempfile.TemporaryFile(mode='w+') as f:
    f.write("временные данные")
    f.seek(0)
    print(f.read())

# Временный файл с именем
with tempfile.NamedTemporaryFile(mode='w+', delete=True) as f:
    print(f.name)  # /tmp/tmpabc123
    f.write("data")

# Временная папка
with tempfile.TemporaryDirectory() as tmpdir:
    path = Path(tmpdir) / "file.txt"
    path.write_text("hello")
    print(list(Path(tmpdir).iterdir()))
```

## CSV файлы

### Чтение CSV
```python
import csv

# Как списки
with open("data.csv", "r", encoding="utf-8") as f:
    reader = csv.reader(f)
    header = next(reader)  # пропустить заголовок
    for row in reader:
        print(row)  # ['Анна', '30', 'Москва']

# Как словари (если есть заголовки)
with open("data.csv", "r", encoding="utf-8") as f:
    reader = csv.DictReader(f)
    for row in reader:
        print(row['name'], row['age'], row['city'])
```

### Запись CSV
```python
import csv

# Как списки
data = [
    ['name', 'age', 'city'],
    ['Анна', 30, 'Москва'],
    ['Боб', 25, 'СПб']
]

with open("out.csv", "w", newline='', encoding="utf-8") as f:
    writer = csv.writer(f)
    writer.writerows(data)

# Как словари
data = [
    {'name': 'Анна', 'age': 30, 'city': 'Москва'},
    {'name': 'Боб', 'age': 25, 'city': 'СПб'}
]

with open("out.csv", "w", newline='', encoding="utf-8") as f:
    writer = csv.DictWriter(f, fieldnames=['name', 'age', 'city'])
    writer.writeheader()
    writer.writerows(data)
```

### Разные диалекты CSV
```python
# Точка с запятой вместо запятой (европейский формат)
with open("data.csv", "r") as f:
    reader = csv.reader(f, delimiter=';')

# Свои настройки
with open("data.csv", "w") as f:
    writer = csv.writer(f, delimiter='|', quotechar='"', 
                        quoting=csv.QUOTE_MINIMAL)
```

```python
import json

data = {
    "name": "Анна",
```

## Работа с бинарными файлами

### Чтение/запись бинарных данных
```python
# Копирование файла
with open("source.jpg", "rb") as src:
    with open("dest.jpg", "wb") as dst:
        dst.write(src.read())  # грузит всё в память!

# Лучше по частям
with open("source.jpg", "rb") as src:
    with open("dest.jpg", "wb") as dst:
        while chunk := src.read(8192):  # читаем по 8KB
            dst.write(chunk)
```

### Работа со структурированными бинарными данными
```python
import struct

# Упаковка чисел в бинарный формат
# fmt: 'i' = int, 'f' = float, 'd' = double, 's' = string
data = struct.pack('iif', 42, 100, 3.14)  # два int, один float

# Распаковка
a, b, c = struct.unpack('iif', data)

# Свои форматы
# '>i' — big-endian int
# '<i' — little-endian int
with open("binary.dat", "wb") as f:
    f.write(struct.pack('>i', 42))
```

## Наблюдение за файловой системой (watchdog)
```python
# Это сторонняя библиотека, но очень полезная
# pip install watchdog

from watchdog.observers import Observer
from watchdog.events import FileSystemEventHandler

class MyHandler(FileSystemEventHandler):
    def on_modified(self, event):
        print(f"Файл изменен: {event.src_path}")
    
    def on_created(self, event):
        print(f"Создан: {event.src_path}")
    
    def on_deleted(self, event):
        print(f"Удален: {event.src_path}")

observer = Observer()
observer.schedule(MyHandler(), path=".", recursive=False)
observer.start()

try:
    while True:
        time.sleep(1)
except KeyboardInterrupt:
    observer.stop()
observer.join()
```

## Сравнение с C++

| Операция | C++ | Python |
|----------|-----|--------|
| Открытие файла | `ifstream file("file.txt")` | `with open("file.txt") as f:` |
| Чтение строки | `getline(file, line)` | `for line in f:` |
| Закрытие | `file.close()` | Автоматически |
| Проверка существования | `filesystem::exists(path)` | `Path(path).exists()` |
| Копирование | `filesystem::copy()` | `shutil.copy()` |
| Пути | `filesystem::path` | `pathlib.Path` |
| Бинарные данные | `file.read(buffer, size)` | `f.read(size)` |

### Безопасная запись (с временным файлом)
```python
def safe_write(filename, content):
    """Запись с временным файлом для атомарности"""
    temp = filename + ".tmp"
    try:
        with open(temp, "w") as f:
            f.write(content)
        os.replace(temp, filename)  # атомарно на большинстве ОС
    except:
        Path(temp).unlink(missing_ok=True)
        raise
```

### Чтение последних N строк файла
```python
def tail(filename, n=10):
    """Читает последние n строк файла"""
    with open(filename, "rb") as f:
        f.seek(0, 2)  # в конец
        filesize = f.tell()
        
        block_size = 1024
        data = []
        lines_found = 0
        
        while filesize > 0 and lines_found <= n:
            read_size = min(block_size, filesize)
            f.seek(-read_size, 1)
            data.append(f.read(read_size).decode())
            lines_found += data[-1].count('\n')
            filesize -= read_size
        
        return ''.join(reversed(data)).splitlines()[-n:]
```
