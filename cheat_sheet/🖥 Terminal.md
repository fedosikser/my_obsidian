## 📍 **Навигация по файловой системе**

```bash
pwd            # текущая директория
ls             # список файлов
ls -la         # все файлы (включая скрытые) с правами
cd <папка>     # перейти в папку
cd ..          # подняться на уровень выше
cd ~           # перейти в домашнюю директорию
cd -           # вернуться в предыдущую директорию
```

---

## 📁 **Управление папками**

```bash
mkdir <имя>        # создать папку
mkdir -p a/b/c     # создать вложенные папки (если их нет)
rmdir <папка>      # удалить **пустую** папку
```

---

## 📄 **Управление файлами**

```bash
touch file.txt     # создать пустой файл
rm file.txt        # удалить файл
rm -r <папка>      # удалить папку с содержимым
rm -rf <папка>     # удалить без подтверждения (❗️ осторожно)
open file          # открыть файл (macOS)
```

---

## 🔄 **Копирование, перемещение, переименование**

```bash
# Копирование
cp file.txt copy.txt          # файл
cp -r folder newfolder        # папка

# Перемещение / переименование
mv file.txt folder/           # переместить
mv old.txt new.txt            # переименовать
mv file.txt dir/new.txt       # переместить + переименовать
```

> [!tip] Важно
> Если при `mv` указать несуществующую папку — файл будет **переименован**, а не перемещён.

## 👀 **Просмотр содержимого файлов**

```bash
cat file.txt          # весь файл сразу
less file.txt         # постранично (q — выход)
head -n 5 file.txt    # первые 5 строк
tail -n 5 file.txt    # последние 5 строк
tail -f file.log      # следить за изменениями (логи)
```

## ✏️ **Редактирование файлов**

### **Nano** (простой)
```bash
nano file.txt
```
- `Ctrl + O` → сохранить
- `Ctrl + X` → выйти

### **Vim** (сложный)
```bash
vim file.txt
```
- `i` → режим вставки
- `Esc` → выйти из режима вставки
- `:wq` → сохранить и выйти
- `:q!` → выйти без сохранения

### **VS Code**
```bash
code file.txt
```

## 🔎 **Поиск**

### **Поиск файлов (find)**
```bash
find . -name "file.txt"          # точное имя
find . -name "*.js"               # все .js файлы
find . -type d -name "src"        # папки с именем src
```

### **Поиск внутри файлов (grep)**
```bash
grep "text" file.txt               # искать в файле
grep -r "text" .                    # рекурсивно в папке
grep -i "text" file.txt             # без учёта регистра
grep -n "text" file.txt             # показать номера строк
```

## ⚙️ **Права доступа**

```bash
ls -l                    # посмотреть права
chmod +x script.sh       # сделать файл исполняемым
chmod 755 script.sh      # установить конкретные права
```

## 🔧 **Управление процессами**

```bash
Ctrl + C        # остановить процесс
Ctrl + Z        # приостановить процесс
jobs            # список фоновых задач
fg              # вернуть задачу на передний план
bg              # запустить фоновую задачу

ps aux          # все процессы
kill <PID>      # убить процесс по ID
kill -9 <PID>   # принудительно убить
```

## 📦 **Полезные мелочи**

```bash
clear           # очистить экран
history         # история команд
which git       # путь к исполняемому файлу
echo $SHELL     # какой шелл используется
alias ll='ls -la'   # создать алиас
```

---

## ⚡ **Горячие клавиши**

- `↑` / `↓` — навигация по истории
- `Tab` — автодополнение
- `Ctrl + A` — в начало строки
- `Ctrl + E` — в конец строки
- `Ctrl + U` — удалить всё до курсора
- `Ctrl + K` — удалить всё после курсора
- `Ctrl + W` — удалить одно слово назад
