# C++ шпаргалка: умные указатели

> [!tip] Зачем нужны?
> Умные указатели автоматически управляют памятью — не нужно вручную вызывать `delete`. **Нет утечек памяти!**

---

## 📊 Сравнение типов

| Указатель | Время жизни | Кто владеет | Когда использовать |
|-----------|-------------|-------------|-------------------|
| `unique_ptr` | один владелец | эксклюзивно | 99% случаев |
| `shared_ptr` | много владельцев | совместно | когда реально нужно делить |
| `weak_ptr` | наблюдает | не владеет | чтобы разорвать циклы |
| `raw ptr` (`T*`) | кто-то другой | не владеет | для наблюдения (never `delete`) |

---

## 🔹 unique_ptr — эксклюзивное владение

```cpp
#include <memory>

// Создание
auto ptr = std::make_unique<int>(42);
std::unique_ptr<MyClass> obj = std::make_unique<MyClass>();

// Использование
*ptr = 100;
obj->doSomething();

// Перемещение (копирование запрещено!)
std::unique_ptr<int> ptr2 = std::move(ptr);  // ptr теперь nullptr

// Освобождение вручную
ptr2.reset();      // удаляет объект
ptr2.reset(new int(10));  // заменяет

// Получить raw-указатель (для работы со старым кодом)
int* raw = ptr2.get();
```

> [!warning] Важно
> `unique_ptr` **нельзя скопировать**, только переместить. Это гарантирует уникального владельца.

---

## 🔹 shared_ptr — совместное владение

```cpp
#include <memory>

// Создание
auto ptr1 = std::make_shared<int>(42);
std::shared_ptr<MyClass> obj = std::make_shared<MyClass>();

// Копирование — счётчик ссылок увеличивается
auto ptr2 = ptr1;  // теперь 2 владельца
auto ptr3 = ptr1;  // теперь 3 владельца

// Сколько владельцев
long count = ptr1.use_count();  // 3

// Освобождение
ptr1.reset();  // счётчик уменьшился до 2
ptr2.reset();  // до 1
ptr3.reset();  // до 0 — объект удалён

// Проверка
if (ptr1) { /* не nullptr */ }
```

## 🔹 weak_ptr — наблюдатель без владения

```cpp
#include <memory>

std::shared_ptr<int> shared = std::make_shared<int>(99);
std::weak_ptr<int> weak = shared;  // не увеличивает use_count()

// Проверить, жив ли объект
if (auto locked = weak.lock()) {   // получаем shared_ptr
    std::cout << *locked << std::endl;  // 99
} else {
    std::cout << "Объект уже удалён" << std::endl;
}

// Проверка без блокировки
bool expired = weak.expired();  // true если объект мёртв
```

## 🎯 Практические правила

| Ситуация | Решение |
|----------|---------|
| **Локальный объект в функции** | `auto ptr = std::make_unique<T>()` |
| **Поле класса (один владелец)** | `std::unique_ptr<T> m_ptr` |
| **Поле класса (делится с другими)** | `std::shared_ptr<T> m_ptr` |
| **Контейнер указателей** | `std::vector<std::unique_ptr<T>>` |
| **Возврат из фабрики** | `std::unique_ptr<T>` или `std::shared_ptr<T>` |
| **Наблюдение без владения** | `T*` (raw) или `weak_ptr` |
| **Разорвать цикл ссылок** | заменить один из `shared_ptr` на `weak_ptr` |

## ⚠️ Ошибки новичков

```cpp
// ❌ НЕПРАВИЛЬНО
MyClass* raw = new MyClass();
std::shared_ptr<MyClass> ptr(raw);  // опасно, но иногда работает
std::shared_ptr<MyClass> ptr2(raw); // УЖАС! double delete

// ❌ ЕЩЁ ОШИБКА
auto ptr = std::make_shared<int>(5);
int* raw = ptr.get();
delete raw;  // НЕЛЬЗЯ! shared_ptr потом сделает double delete

// ✅ ПРАВИЛЬНО
auto ptr = std::make_shared<int>(5);
int* raw = ptr.get();  // только наблюдать, не удалять
```

## 🔄 Циклические ссылки (память не освободится)

```cpp
struct Node {
    std::shared_ptr<Node> next;  // ❌ создаёт цикл
    ~Node() { std::cout << "Deleted\n"; }
};

auto a = std::make_shared<Node>();
auto b = std::make_shared<Node>();
a->next = b;
b->next = a;  // цикл! деструкторы не вызовутся

// ✅ ИСПРАВЛЕНИЕ
struct Node {
    std::weak_ptr<Node> next;    // weak_ptr разрывает цикл
    // ... остальное так же
};
```

## 🚀 Советы для реальных проектов

```cpp
// 1. Всегда используйте make_unique/make_shared
auto good = std::make_unique<MyClass>(arg1, arg2);     // ✅
auto bad = std::unique_ptr<MyClass>(new MyClass(arg1, arg2));  // ⚠️

// 2. Для массивов (C++17 и выше)
auto arr = std::make_unique<int[]>(100);   // unique_ptr<int[]>
arr[0] = 42;

// 3. Кастомный удалятор
auto file = std::unique_ptr<FILE, decltype(&fclose)>(
    fopen("test.txt", "r"), &fclose
);

// 4. Приведение типов
std::shared_ptr<Base> base = std::make_shared<Derived>();
auto derived = std::dynamic_pointer_cast<Derived>(base);
```
